# PneumoniaAI
PneumoniaAI: A full-stack AI system for detecting pneumonia from chest X-rays using ensemble deep learning (DenseNet, ResNet, EfficientNet) with explainable AI (Grad-CAM) and FastAPI backend.


AI-powered system for detecting pneumonia from chest X-rays using ensemble deep learning.

## Features

* Ensemble models (DenseNet121, ResNet50, EfficientNet)
* Grad-CAM explainability
* Severity classification (MILD, MODERATE, SEVERE)
* FastAPI backend

## How to Run

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## Note

Model files and environments are not included due to size limitations.
